# Clase 04

## Ramas (Branches)

![ramas](https://miro.medium.com/max/801/1*DhagidpZutkaCmAZobmzDQ.png)

### Crear una rama

```sh
git branch <nombre-rama>
git branch data-ramas
```

### Listar una rama

```sh
git branch
git branch
```

### Cambiar entre ramas

```sh
git switch <nombre-rama>
git switch data-ramas
```
